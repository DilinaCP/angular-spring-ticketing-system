
# Real-Time Event Ticketing System

Real-Time Ticket Booking System Introduction This is a real-time ticket booking system where vendors can release tickets, and customers can retrieve them from a common pool. The system is designed to handle concurrent requests and ensure that tickets are managed efficiently. The application consists of three main components: • Frontend: Built with Angular, providing a user interface for configuring and monitoring the system. • Backend: Built with Spring Boot, handling the core logic, and providing a WebSocket API for real-time updates..
